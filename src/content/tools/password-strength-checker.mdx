## Password Security in 2026

With data breaches exposing billions of credentials every year, strong passwords are your first line of defense. Yet studies show that **over 80% of breaches** involve weak or reused passwords.

### How Passwords Get Cracked

| Attack Method | Speed | Effective Against |
|---|---|---|
| **Brute force** | 10B guesses/sec (GPU) | Short passwords (under 8 chars) |
| **Dictionary attack** | Millions of common words | Passwords using real words |
| **Credential stuffing** | Automated login attempts | Reused passwords |
| **Rainbow tables** | Instant lookup | Unsalted hashes |
| **Social engineering** | Varies | Passwords based on personal info |

### Password Strength Tiers

- **Very Weak (0-20)**: Can be cracked in seconds. Common patterns like "password123" or "qwerty"
- **Weak (20-40)**: Cracked in minutes to hours. Too short or missing character types
- **Fair (40-60)**: Cracked in days to months. Decent but could be improved
- **Strong (60-80)**: Would take years to crack. Good mix of length and complexity
- **Very Strong (80-100)**: Would take centuries or longer. Excellent security

### Best Practices for Password Security

1. **Use a unique password for every account** — if one is breached, others stay safe
2. **Aim for 12+ characters** — length matters more than complexity
3. **Use a password manager** — Bitwarden, 1Password, or KeePass
4. **Enable 2FA everywhere** — even strong passwords can be phished
5. **Consider passphrases** — "purple-elephant-dancing-moonlight" is both strong and memorable
6. **Never share passwords** — not even with IT support (legitimate IT will never ask)

### The Math Behind Password Strength

Password entropy is calculated as: **E = L × log₂(C)**

Where:
- **L** = password length
- **C** = character set size (26 for lowercase, 52 with uppercase, 62 with digits, 95 with specials)

A password "aB3$" has only 4 × log₂(95) ≈ 26 bits of entropy — crackable in milliseconds.
A password "correct-horse-battery-staple" has ~44 bits per word × ~4 = ~132 bits — virtually uncrackable.
